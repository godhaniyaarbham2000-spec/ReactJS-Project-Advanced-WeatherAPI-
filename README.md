# 🌦️ React Weather App - Weather Forecast Project

A modern **React + Vite Weather Application** that fetches real-time weather data using APIs.  
Built for learning API integration, React hooks, and UI rendering for real-world use cases.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌍 Auto location detection using geocoding API
* 🌡️ Real-time temperature display
* 💧 Humidity tracking
* 📉 Min & Max temperature forecast
* 🌦️ Dynamic weather condition UI (Sunny / Rain / Winter)
* 🖼️ Weather-based background images
* ⚡ Instant UI update without refresh

---

## 🛠️ Tech Stack

* React.js (Hooks: useState)
* Vite (Fast build tool)
* JavaScript (ES6+)
* Material UI (MUI Components)
* Open-Meteo API (Weather Data)
* Open-Meteo Geocoding API
* CSS3

## ⚙️ How to Run This Project

### 1. Clone the repository
```
git clone https://github.com/your-username/react-weather-app.git
```

### 2. Move to project folder
```
cd react-weather-app
```
### 3. Install dependencies
```
npm install
```

### 4. Start development server
```
npm run dev
```
